Database querying and persistence.

ransaq stores crawled data in SQLite using sqlx.

Setup

To set up a development database you will need to install sqlx-cli,

cargo install sqlx-cli \
    --no-default-features \
    --features sqlite \
    --features rustls

create a .env file with a DATABASE_URL,

DATABASE_URL=sqlite:ransaq.sqlite

and finally run

sqlx database setup

Schema

You can view the schema by looking through the /migrations directory, or by using the sqlite1 command as follows:

sqlite3 -readonly ransaq.sqlite ".schema"

  1. You will need to be running SQLite version 3.37.0 or later due to the use of STRICT tables (https://www.sqlite.org/releaselog/3_37_0.html) ↩
